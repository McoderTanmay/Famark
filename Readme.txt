Famark Cloud Tutorial for Node JS

Before running the app.js make sure that you have done the following

-Have basic Understanding of creating entities,records and their attributes in famark cloud 
-Have besic understanding of Javascript
-Have installed Node js in your system

before running app.js do the following steps
Run commands
-npm i prompt-sync
-npm i nodemon
-npm i express
And add "start": "nodemon app.js", in package.json
